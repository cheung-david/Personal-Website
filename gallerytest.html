David's Portfolio Software Projects

Android - LibGDX - Elemental Jumper
Developed an open source 2D jumping game for Android using Java and LibGDX as the API. Published on Google Play.
Android - Random Chooser
Developed an Android app that allows users to enter choices and randomly selects one for them. Option of generating X list of random numbers given a range. Ability to save and load choice list to external storage. Published on Google Play.
C++ - Maze Game - SFML 2.3
Developed a basic maze game using the SFML graphics library. The goal is to move the cyan player to the green box. The maze is generated using a randomized depth-first search recursive backtrack algorithm. The path finder is implemented using breadth first search on a grid based map system.
C++ - Autocompleter
Developed an efficient word auto completer using a prefix tree (Trie). This program will find all auto completions in memory for any given prefix.
C++ - Freeroam Game - OpenGL (on-going)
Worked with a partner to develop a 2D top-down open-world game, similar to Grand Theft Auto.
